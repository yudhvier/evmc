1 # EVMC – Ethereum Client-VM Connector API {#mainpage}
2 
3 **ABI version 7**
4 
5 The EVMC is the low-level ABI between Ethereum Virtual Machines (EVMs) and
6 Ethereum Clients. On the EVM-side it supports classic EVM1 and [ewasm].
7 On the Client-side it defines the interface for EVM implementations
8 to access Ethereum environment and state.

17 # Versioning {#versioning}
18 
19 The EVMC project uses [Semantic Versioning](https://semver.org).
20 The version format is `MAJOR.MINOR.PATCH`.
21 
22 The _MAJOR_ version number is also referenced as the **EVMC ABI version**.
23 This ABI version is available to VM and Host implementations by ::EVMC_ABI_VERSION.
24 For example EVMC 3.2.1 would have ABI version 3 and therefore this project release
25 can be referenced as EVMC ABIv3 or just EVMC 3.
26 Every C ABI breaking change requires increasing the _MAJOR_ version number.
27 
28 The releases with _MINOR_ version change allow adding new API features
29 and modifying the language bindings API.
30 Backward incompatible API changes are allowed but should be avoided if possible.
31 
32 The releases with _PATCH_ should only include bug fixes. Exceptionally,
33 API changes are allowed when required to fix a broken feature.

66 ## Terms
67 
68 1. **VM** – An Ethereum Virtual Machine instance/implementation.
69 2. **Host** – An entity controlling the VM.
70  The Host requests code execution and responses to VM queries by callback
71  functions. This usually represents an Ethereum Client.

74 ## Responsibilities
75 
76 ### VM
77 
78 - Executes the code (obviously).
79 - Calculates the running gas cost and manages the gas counter except the refund
80  counter.
81 - Controls the call depth, including the exceptional termination of execution
82  in case the maximum depth is reached.
83 
84 
85 ### Host
86 
87 - Provides access to State.
88 - Creates new accounts (with code being a result of VM execution).
89 - Handles refunds entirely.
90 - Manages the set of precompiled contracts and handles execution of messages
91  coming to them.
